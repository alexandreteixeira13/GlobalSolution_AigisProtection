# Aigis Protection

Landing page desenvolvida para a Global Solution FIAP 2026, com foco no uso da indústria espacial para resolver problemas reais na Terra.

A Aigis Protection propõe o uso de dados de satélites, clima, ambiente e saúde pública para prever riscos epidemiológicos em regiões remotas. A ideia é ajudar comunidades ribeirinhas, indígenas e rurais a receberem ações preventivas antes que surtos biológicos cresçam.
